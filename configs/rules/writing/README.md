# Writing Rules

Conventions for producing text artefacts: markdown documentation, LaTeX, and scientific prose.

---

## Files

| File | Covers |
|---|---|
| `docs.md` | Markdown documentation: characters, formatting, structure, tables, code blocks, caveats |
| `latex.md` | LaTeX formatting: cross-references, labels, math layout, footnotes |
| `science.md` | Scientific writing: argument structure, physical interpretation, mathematical writing, grammar, rhetoric |
