# Scientific Writing Style

These rules apply when writing or editing scientific prose.

---

## Core Philosophy

| Rule | |
|---|---|
| Build arguments causally | each paragraph should explain why the next idea follows |
| Prioritise physical interpretation | formalism supports the physics, not vice versa |
| Motivation before machinery | explain why something matters before introducing equations or methods |
| Significance emerges naturally | avoid announcing importance rhetorically |
| Maintain conceptual continuity | connect equations, simulations, and observations into a single narrative |
| Preserve scale awareness | distinguish local processes from system-scale implications |

The writing should feel physically motivated rather than procedurally descriptive. Exposition should progressively connect mechanisms, limitations, numerical approaches, and astrophysical consequences.

---

## Sentence Structure

| Rule | |
|---|---|
| Declarative sentences | avoid rhetorical questions |
| Dense but controlled | each sentence should contribute new information |
| Active causal transitions | prefer "because", "therefore", "this matters because", "to overcome this" |
| Minimal hedging | qualify uncertainty precisely rather than generically |
| Prefer mechanisms over labels | explain what physically changes and why |
| Avoid filler transitions | cut "interestingly", "notably", "it is important to note" unless necessary |
| Avoid dramatic framing | maintain restrained exposition |
| Use contrast carefully | prefer "while X..., Y..." and "this can..., but does not by itself..." |

---

## Paragraph Structure

| Rule | |
|---|---|
| One conceptual progression per paragraph | avoid mixing unrelated ideas |
| Opening sentence establishes focus | define the physical or conceptual problem immediately |
| Middle sentences develop mechanism | elaborate consequences, limitations, or interpretation |
| Final sentence broadens scope | reconnect to observations, simulations, or larger implications |
| Transitions should be causal | the next paragraph should emerge naturally from unresolved consequences |

Paragraphs should function as logical progressions, not collections of related statements. The reader should be able to follow why each sentence appears where it does.

---

## Scientific Framing

| Rule | |
|---|---|
| Introduce tensions explicitly | frame unresolved or incomplete aspects of the problem |
| Explain why limitations matter | do not merely state assumptions or caveats |
| Separate mechanism from implication | first explain what happens, then why it matters |
| Distinguish evidence from interpretation | numerical results and physical claims should remain separable |
| Bound conclusions carefully | avoid implying broader claims than directly supported |
| Reconnect abstractions to observables | relate formal discussion back to measurable or physical consequences |

Scientific motivation should emerge through physical tensions, limitations, or unexplained behaviour, rather than through rhetorical claims of importance.

---

## Mathematical Exposition

| Rule | |
|---|---|
| Introduce physical intuition before notation | define the process before the symbols |
| Equations should advance the argument | avoid decorative or isolated mathematics |
| Decompose equations physically | explain what terms represent dynamically |
| Connect scales explicitly | relate formal results back to physical length or time scales |
| Avoid equation dumping | each equation should have narrative purpose |
| Derivations should preserve interpretation | mathematics should not become detached from the physics |

Mathematics should function as part of the explanatory narrative. The reader should understand not only what an equation says, but what physical process it represents.

---

## Interpretation Style

| Rule | |
|---|---|
| Physical interpretation accompanies results | numerical behaviour should not stand alone |
| Mechanistic explanations preferred | explain processes rather than categories |
| Preserve uncertainty structure | identify unresolved components explicitly |
| Avoid overstating causality | distinguish correlation, mechanism, and interpretation |
| Local and global implications should remain distinct | specify which conclusions survive scale transitions |

The prose should remain analytically restrained. Conclusions should feel earned through the preceding logic rather than asserted through emphasis.

---

## Scale Handling

| Rule | |
|---|---|
| Track relevant scales explicitly | identify dissipation, inertial, driving, and system scales where relevant |
| Clarify scale transitions | explain when conclusions do or do not survive coarse-graining |
| Connect local physics to global implications | bridge simulation-scale results to astrophysical interpretation |
| Preserve hierarchy awareness | distinguish microphysical effects from emergent system behaviour |

Scale transitions are often where the physical meaning of a result changes. The writing should make these transitions explicit rather than implicit.

---

## Rhetorical Restraint

| Rule | |
|---|---|
| No hype language | avoid "remarkable", "groundbreaking", "extremely important" |
| No performative novelty claims | novelty should emerge from comparison and reasoning |
| No dramatic reveals | maintain steady exposition |
| No exaggerated certainty | avoid stronger claims than the evidence supports |
| Avoid excessive signposting | structure should emerge naturally from the prose |

The tone should remain controlled and technically grounded. The writing should sound confident because the reasoning is coherent, not because the language is emphatic.

---

## Preferred Patterns

Use contrastive structure to organise reasoning:

```text
While X..., Y...
This can..., but does not by itself...
Necessary to..., however...
```

Use consequence-driven transitions:

```text
This matters because...
An immediate consequence of this is...
To overcome this...
This introduces...
This suggests...
```

Use scale-aware framing:

```text
At local scales...
At system scales...
This modifies the characteristic scale...
This remains insufficient to explain...
```

---

## Things To Avoid

| Rule | |
|---|---|
| No disconnected textbook exposition | maintain narrative continuity |
| No abstraction without grounding | reconnect concepts to physical systems |
| No purely procedural derivations | preserve physical interpretation |
| No empty framing statements | cut sentences that only announce structure |
| No excessive literature summarising | literature discussion should support the argument |
| No inflated conclusions | preserve proportionality between evidence and claim |

---

## Canonical Style Characteristics

The prose should feel:

- physically motivated rather than formally procedural,
- analytically restrained rather than rhetorically emphatic,
- dense without becoming opaque,
- interpretive without becoming speculative,
- technically precise while remaining readable,
- conscious of scale, limitation, and consequence.
